## 数学趣味游戏

### 游戏列表

**速算 / 基础**

| 游戏 | 玩法 |
|------|------|
| 24 点 | 用 4 张牌（A=1，J=11，Q=12，K=13）通过加减乘除凑出 24，越快越好 |
| 心算闯关 | 60 秒内连续口算，答对题数越多越好 |
| 猜数字 | 猜出 1–100 的神秘数字，次数越少越好 |
| 质数猎人 | 30 秒快速判断质数，连对有加成 |
| 乘法闪卡 | 限时回答乘法表，正确率决定得分 |

**数字 / 逻辑**

| 游戏 | 玩法 |
|------|------|
| 数字猜谜 | 猜出 4 位不重复数字，位置和数字都要对（Mastermind） |
| 河内塔 | 将所有圆盘从左柱移到右柱，小盘不能压大盘，步数越少越好 |
| 谁说真话 | 根据陈述推理谁是骑士（只说真话）、谁是无赖（只说假话） |
| 数字华容道 | 滑动方块，将 1–15 按顺序排好，步数越少越好 |
| 幻方填数 | 填入数字使每行、每列、每条对角线之和相等 |
| 填运算符 | 在数字间填入 +-×÷ 使等式成立 |
| 记忆矩阵 | 短暂记忆数字矩阵的位置，之后复现 |

**速算 / 估算**

| 游戏 | 玩法 |
|------|------|
| 找规律 | 找出数列规律，填入下一项 |
| 二进制转换 | 60 秒在十进制与二进制之间快速互转 |
| 分数比大小 | 30 秒内快速判断两个分数哪个更大 |
| 因数猎手 | 找出给定数的所有因数 |
| 速算接龙 | 前一题的答案是下一题的起点，看能接多长 |
| 角度估算 | 目测角的度数，越准确得分越高 |
| 坐标射击 | 根据坐标，在坐标系中精准点击目标 |

**趣味科普**

| 游戏 | 玩法 |
|------|------|
| 天平称假币 | 用最少次数的称重找出那枚假币 |

### 功能特性

- 成绩保存在本地（localStorage），刷新不丢失
- 主界面支持列表 / 图标两种视图，可查看所有游戏历史记录
- 支持用户反馈和微信打赏

## 技术栈

- 纯 H5（HTML + CSS + JavaScript），零框架依赖
- nginx 提供静态文件服务
- Docker（nginx:alpine）容器化

## 本地开发

```bash
# 启动本地预览（需要 Docker）
make local
# 访问 http://localhost:8080

# 停止
make stop
```

## 部署

目标环境：AWS Lightsail VPS（nginx，Ubuntu）

### 首次初始化

1. 复制并填写配置文件：

   ```bash
   cp deploy.config.example deploy.config   # 如有模板
   # 编辑 deploy.config，填写以下字段：
   #   LIGHTSAIL_HOST  服务器公网 IP
   #   SSH_USER        SSH 用户名
   #   SSH_KEY_PATH    本地密钥文件路径（.pem）
   #   REMOTE_PATH     服务器部署目录（默认 /var/www/apps）
   ```

2. 初始化服务器（仅首次，安装 nginx）：

   ```bash
   make setup
   ```

### 日常部署

```bash
make deploy
```

脚本会自动完成：rsync 同步文件 → 更新 nginx 配置 → reload nginx。
